## 3. Production Instructions — AI Video Tools

Pick **one** of the three tools below. All can produce a finished 60-second video in under 30 minutes from this script.

### Option A — HeyGen (AI avatar speaking the script) ⭐ Recommended for solo founders

1. Sign up at **heygen.com** (free tier supports 1-min videos).
2. Create new video → **"Avatar Video"**.
3. Choose a professional avatar (or upload your photo for **Instant Avatar** — paid tier).
4. Paste the **Final Script** (Section 1 above) into the script box.
5. Voice: select an English voice with neutral accent (e.g., "Daniel — Professional").
6. **Add scene breaks every 6–10 seconds** using the scene editor; for each scene, upload the on-screen visuals from the table above (use Canva or your existing pitch deck slides exported as PNG).
7. Background: pick a clean office/studio template.
8. Export at **1080p MP4**.
9. Drop the file into `public/elevator-pitch.mp4` and embed on your site.

### Option B — Pictory (text-to-video with stock b-roll)

1. Sign up at **pictory.ai**.
2. Create → **"Script to Video"**.
3. Paste the **Final Script**.
4. Pictory auto-suggests b-roll per sentence — replace any irrelevant clips with: tech office, AI dashboard, charts, handshake, world map.
5. Voiceover: enable AI voiceover (Pictory's "Liam" or "Olivia" works well).
6. Add background music: pick "Corporate Inspiring" at 15% volume.
7. Add captions (auto-generated, branded in NovusSparks Cyan #00BFFF).
8. Add intro/outro using your NovusSparks logo (`src/assets/images/novussparks-icon.svg`).
9. Export 1080p MP4.

### Option C — Canva (slide-style video, best for design control)

1. Open **canva.com** → search **"Pitch Video"** template, 1920×1080.
2. Create **8 slides** matching the scene table above; copy the on-screen visuals exactly.
3. Set each slide's duration per the timing column (8s, 12s, 12s, 7s, 7s, 5s, 5s, 4s).
4. **Voiceover options:**
   - Record yourself reading the script directly in Canva (Upload → Record).
   - OR use **ElevenLabs** (elevenlabs.io) — paste script, pick a voice ("Adam" or "Rachel"), download MP3, upload to Canva.
5. Add subtle slide transitions ("Fade", 0.3s).
6. Add background music from Canva's library (Corporate / Inspiring).
7. Download as **MP4 video**.

## 4. Recording Tips (if you record yourself instead)

- **Camera at eye level**, ring light or window in front.
- **Read each section in 1 take**; cut the best take of each later.
- **Practice 3 times** to hit the 60-second mark — script is timed for ~150 wpm.
- **Smile during the intro and ask** — it changes vocal energy.
- **Record vertical AND horizontal** versions if you'll post on LinkedIn + TikTok.
